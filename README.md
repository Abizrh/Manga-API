# Manga-API

REST API manga berbahasa indonesia, didapatkan dengan scraping dari web komikindo.co dan dibuat dengan node.js.

## Fitur

* Cari manga
* List manhua, manga, manhwa
* Genre
* Baca chapter
* Update terbaru
* Detail Manga
